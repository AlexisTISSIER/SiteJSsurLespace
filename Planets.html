PLANETE

Le fond que vous voyez est une image capturée par le télescope de la NASA.
Cette image est automatiquement mise à jour chaque jour à une heure précise grâce à une API et une clé API fournie par la NASA.
Pour obtenir cette clé API, j'ai utilisé le site web https://api.nasa.gov/.
Cette plateforme m'a permis de générer une clé API unique qui me donne accès à divers services et données de la NASA.
L'URL que j'utilise pour récupérer l'image de fond est https://api.nasa.gov/planetary/apod?api_key=${maCleAPI}.
Cette URL spécifique me permet d'appeler l'API de la NASA et de récupérer l'image du jour à afficher en arrière-plan de la page.
